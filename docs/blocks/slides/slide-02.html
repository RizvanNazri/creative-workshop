Концепция

Мастерская Курорта представляет собой территорию творчества с элементами изготовления духовых инструментов из бамбука, таких как флейты, диджериду, а также картин с использованием техники нитяной графики. В дальнейшем предусматривается оборудование гончарными станками для производства глиняных изделий. Начальный перечень услуг, предоставляемых в мастерской: 1. Обучение игре на духовых инструментах 2. Мастер классы по нитяной графике и изготовление картин 3. Изготовление флейт пимак и диджериду из бамбука 4. Проведение чайных церемоний и звукотерапии, медитация 5. Изготовление рисунков сакральной геометрии с возможностью нанесения их на одежду. Каждая услуга будет направлена на развитие и раскрытие творческих способностей. Изготовленные изделия будут реализовываться как сувениры. Кроме того, каждый гость сможет участвовать в процессе создания изделий. В дальнейшем возможно привлечения мастеров и других направлений для расширения спектра услуг Мастерской Курорта. Для старта проекта Мастерская Курорста / Ра: Потребуется отдельно стоящее помещение 60-80 кВ.м, оборудование для мастерской (станки, инструменты, столы) общей стоимостью около 500 тыс. руб.
